# 常用指标

## 容量与限制

| 类型 | 最大限制 |
|------|----------|
| 节点最大个数 | **512** |
| 单节点最大连接数 | **10,000** |
| 单存储空间中数据文件最大个数 | **32,767** |
| 用户数据库的最大个数 | **32,767** |
| 单库内表的最大个数 | **21 亿** |
| 表中字段最大个数 | **2,040** |
| 常规表最大记录条数 | **无限制** |
| 常规记录的最大长度 | **64KB** |
| 连接查询的最大表个数 | **64** |
| 索引定义最大长度 | **2,000B** |
| 大对象最大长度 | **2GB** |
| SQL 命令最大长度 | **2MB** |
| SQL 命令参数最大个数 | **2,047** |

## 支持的接口语言

JDBC / Python / C# / Go / PHP / Node.js / ODBC / NCI / OCI / XGCI等

## 兼容模式

| 项目 | 支持 |
|------|------|
| 异构库兼容模式 | Oracle / MySQL / PostgreSQL |
| SQL 标准 | SQL-92、SQL-99、SQL-2003 |

## 与主流数据库限制对比

| 指标 | XuguDB | Oracle | MySQL | PostgreSQL |
|------|--------|--------|-------|------------|
| 最大节点数 | 512 | 128 (RAC) | 无原生集群 | 无原生集群 |
| 单节点连接数 | 10,000 | 依赖配置 | 依赖配置 | 依赖配置 |
| 单库表数量 | 21 亿 | 无明确限制 | 无明确限制 | 无明确限制 |
| 字段最大个数 | 2,040 | 1,000 | 4,096 | 1,600 |
| 记录最大长度 | 64KB | 依赖块大小 | 65,535B | 1.6TB |
| JOIN 表个数 | 64 | 无明确限制 | 61 | 无明确限制 |
| 索引键最大长度 | 2,000B | 约 6,400B | 3,072B | 2,712B |
| 大对象最大 | 2GB | 4GB | 4GB | 1GB (TOAST) |
| SQL 最大长度 | 2MB | 无明确限制 | 1GB (max_allowed_packet) | 无明确限制 |
