# 产品简介

## 产品定位

虚谷数据库（XuguDB）是成都虚谷伟业科技有限公司基于 20 多年数据库领域研发经验的技术沉淀，自主原创的高性能关系型数据库管理系统。

### 核心优势

| 优势 | 说明 |
|------|------|
| **安全可靠的国产化根基** | 深度契合国产化部署需求，提供全方位数据安全保障，满足关键行业对数据主权与合规性的严格要求 |
| **多源兼容与架构可塑性** | 全面支持 SQL 国际标准语法，兼容主流关系型数据库应用场景，同时支持单机集中式、集中式集群（主备/双活）及分布式集群架构 |
| **面向多核的性能突破** | 针对现代多核处理器架构深度优化，在高并发、大数据量场景下展现卓越性能 |

### 关键特性

- **通用型**：兼容 SQL-2003 标准及 Oracle/MySQL/PostgreSQL 语法，兼容复杂 SQL、存储过程、触发器、外键等高级功能
- **透明化**：智能分片功能，无需人工指定分片键即可实现自动分片和存储均衡，节点存储倾斜率控制在 5% 以内；支持在线弹性扩容（1→N 节点），扩容后数据自动重新均衡分布
- **高性能**：QPS 可达百万，TPC-C 基准测试单机 TPMC 值可达百万级；在快速定位检索、索引/非索引混合查询、复杂 JOIN、分组统计、高并发等场景专项优化

## 与主流数据库对比

| 特性 | XuguDB | Oracle | MySQL | PostgreSQL |
|------|--------|--------|-------|------------|
| SQL 标准 | SQL-92/99/2003 | SQL-92/99/2003 | SQL-92 部分 | SQL-2011 |
| 分布式架构 | 原生 Shared-Nothing | RAC (Shared-Everything) | 需中间件 (Vitess等) | Citus 插件 |
| 智能分片 | 内置自动分片 | 需手动配置 | 需中间件 | 需 Citus 配置 |
| 在线弹性扩容 | 原生支持，运维零干预 | RAC 扩节点 | 依赖中间件 | 依赖 Citus |
| 国产化适配 | 全面适配国产硬件/OS | 不支持 | 部分支持 | 部分支持 |
| 安全认证 | EAL4+/国密/涉密 | EAL4+ | 无 | 无 |

## 产品认证

### 自主原创认证
- **软件著作版权**：中国版权保护中心虚谷数据库管理系统系列产品软件著作版权（标准版、企业版、分布式版、安全版）
- **产品自主原创**：内核关键模块通过中国信息安全评测中心自主原创审查
- **代码自主扫描**：通过工信部赛普评测中心产品源代码自主性扫描，核心代码自有率达 **99.6%**

### 金融行标认证
- 通过人民银行下属北京国家金融科技认证中心分布式数据库检测，获得分布式数据库金融标准验证证书

### 安全涉密认证
- 经国家保密科技测评中心检测，符合《涉及国家秘密的信息系统安全数据库产品技术要求》
- 安全等级达到国家信息安全等级保护要求的 **EAL4+** 级

### 质量与服务认证
- ISO 9001 质量体系认证
- ISO 27001 信息安全管理体系认证
- ITSS 信息技术服务认证

## 遵循标准

| 标准类别 | 标准名称 | 说明 |
|----------|----------|------|
| 数据库 SQL 标准 | SQL-92、SQL-99、SQL-2003 | 关系数据库标准语言 |
| 数据库通用标准 | 《集中式/分布式数据库政府采购需求标准》 | 中国电子工业标准化技术协会制定 |
| 数据库应用标准 | JR/T0204—2020 分布式数据库技术金融应用规范 | 中国人民银行发布 |
| 中文字符集标准 | GB18030-2022（最高 3 级认证） | 国家标准中文字符集 |
| 商用密码标准 | GM/T0028、GB/T38636 | 密码模块安全达国家安全二级 |
| 信息安全标准 | GB/T20273、GB/T18336 | 安全等级 EAL4+ |
