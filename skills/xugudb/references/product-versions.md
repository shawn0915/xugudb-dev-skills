# 产品版本

## 版本概述

虚谷数据库产品系列包含 4 个版本：

| 版本 | 定位 | 集群规模 | 适用场景 |
|------|------|----------|----------|
| **标准版** | 中小规模业务 | 单机 | 网站建设、政务内网、移动应用、办公系统、管理系统 |
| **企业版** | 中型规模业务 | 双机 | 关键核心业务系统、大规模并发用户访问、海量数据管理 |
| **分布式版** | 大数据应用 | 3~N 机 | GB~PB 级海量大数据应用、OLTP/OLAP/HTAP 混合业务 |
| **安全版** | 高安全领域 | 单机/双机 | 高保密、数据敏感、安全防护要求较高的业务系统 |

## 版本详细说明

### 标准版
- 具备数据库基本的对象管理、安全管理、事务特性、备份恢复与应用开发功能
- 价格较为低廉，具有较高的使用性价比
- 数据管理规模与并发访问用户数相对有限
- 可根据需求平滑升级至企业版

### 企业版
- 扩充了多核并行运算、TB 级海量数据存管、高并发量等重载应用场景支持
- 部署形态支持**双机热备**模式
- 覆盖绝大部分数据库使用场景

### 分布式版
- 包含分布式 SQL 引擎、分布式事务、分布式存储、分布式计算等原生分布式框架
- 具备高性能、高安全、高可靠、高扩展等核心服务能力
- 性能并发近线性增长
- 具备 OLTP、OLAP 及 HTAP 混合型业务应用能力
- 支持主备、两地双中心、多地三中心等部署模式

### 安全版
- 符合国家信息安全标准，安全级别达到 **B1** 级要求
- 多维度加密、基于标记强制安全访问、细粒度操作行为审计
- 从**访问安全、数据安全、运行安全、管理安全、审计安全**五个维度守护数据安全

## 版本选型建议

```
业务规模小、预算有限 → 标准版
  ↓ 需要高可用或大规模并发
关键核心业务、双机热备 → 企业版
  ↓ 需要水平扩展或 PB 级数据
分布式集群、HTAP 混合 → 分布式版
  ↓ 有安全合规要求
高保密、涉密认证要求 → 安全版
```
